# My Website

Hosted on GitHub Pages.

## Deploy

1. Create a new repo on GitHub named `my-website` (no README).
2. Run in PowerShell:
   ```powershell
   .\deploy-to-github.ps1 -RepoUrl "https://github.com/YOUR_USERNAME/my-website.git"
   ```
3. In the repo: **Settings → Pages →** Deploy from branch **main** → Save.

Site URL: `https://YOUR_USERNAME.github.io/my-website/`
